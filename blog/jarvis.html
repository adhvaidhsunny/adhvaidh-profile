The Challenge

At Cal Hacks, UC Berkeley's premier hackathon with over 3,300 participants, we tackled the challenge of making technology accessible to everyone ‚Äî particularly those with visual impairments and motor disabilities.

Our Solution

Jarvis is a voice-powered AI agent that enables users to navigate websites, fill forms, and complete online tasks entirely through natural language. Unlike traditional screen readers, Jarvis understands context and can intelligently interact with modern web applications.

  • Natural language commands for all web interactions
  • Real-time voice feedback using ElevenLabs
  • Smart form filling with context awareness
  • Seamless integration with any website
